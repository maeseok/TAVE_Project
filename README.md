# Tave_Project
웹툰 실사화 배우 추천 시스템
## 👨‍🏫 프로젝트 소개
최근 비질란테, 이두나 등 웹툰을 원작으로 실사화한 드라마가 이슈화됨에 따라 트렌드를 따라가는 주제 선정으로, 더 높은 유사도의 배우 캐스팅과 현재 실사화 되지 않은 웹툰의 궁금증 해결을 목표로 함.

## ⏲️ 개발 기간 
- 2023.11.11(토) ~ 2023.01.27(토)
- 주제 선정
- 데이터 수집
- 데이터 전처리
- 모델링
- 웹 페이지 구현
- 발표
